## 知识回顾 - 数据的流程 1

三种情况：

1. 顺序执行，无任何判断

2. 有判断，分情况执行不同的语句

3. 遇到循环条件，循环执行循环内的语句，直到条件不满足退出循环

`练习 1 ：`

**输出 100 个 hello world**

```js
for (var i = 0; i < 100; i++) {
  console.log(i + 1 + " hello world");
}
```

**结果**

```bash
hello world * 100
```

`练习 5 ：`

**输出 1-100 的所有奇数**

```js
// 法一：
for (var i = 1; i <= 100; i += 2) {
  console.log(i);
}
// 法二：
for (var i = 0; i <= 100; i++) {
  if (i % 2 !== 0) {
    // 一定要用 严格不等
    console.log(i);
  }
}
```

**结果**

```bash
1
3
5
7
9
···
97
99
```

`练习 6 ：`

**定义一个数组，输出数组中所有的奇数**

```js
// 正常情况：
var nums = [23, 32, 453, 11, 67, 34];

for (var i = 0; i < nums.length; i++) {
  if (nums[i] % 2 !== 0) {
    console.log(nums[i]);
  }
}
```

**结果**

```bash
23
453
11
67
```

```js
// 当数组中包含其他类型的字面量？
var arr = [123123, 123545, 12, 56, 342, 213123, true, "asdad"];
for (var i = 0; i < arr.length; i++) {
  var toBNum = +arr[i] || 0; // 非数字就转为数字，NaN就转为0
  if (arr[i] % 2 !== 0 && toBNum) {
    console.log(arr[i]);
  }
```

`练习 7 ：`

**定义一个数组，找出所有的奇数，放入到一个新数组中**

```js
var oldNums = [23, 32, 453, 11, 67, 34];
var newNums = [];

for (var i = 0; i < oldNums.length; i++) {
  if (oldNums[i] % 2 !== 0) {
    newNums.push(oldNums[i]);
  }
}

console.log(newNums);
```

**结果**

```bash
[ 23, 453, 11, 67 ]
```
